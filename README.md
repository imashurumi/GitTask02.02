# GitTask02.02
Практическая работа по МДК 02.02
#Состав репозитория
Состоит из практических работ по МДК 02.02 начиная от:
-ЛР 2.1 - Разработка структуры проекта
-ЛР 2.2 - Разработка модульной структуры проекта (диаграммы модулей)
-ЛР 2.3 - Разработка перечня артефактов и протоколов проекта

#Описание каждой ЛР
-ЛР 2.1 - Разработка структуры проекта для маленького приложения SimpleNote где описаны папки, логика и создан макет приложения.
-ЛР 2.2 - Разработка модульной структуры проекта по теме "Генератор заметок" связанная с предыдущей практической работой приложения SimpleNote. Создана диаграмма приложения по принципу работы приложения.
-ЛР 2.3 - Разработка перечня артефактов и протоколов проекта по теме "Генератор заметок" связанная с предыдущей практической работой. Созданы артефакты и протоколы в процессе разработки программного проекта.
